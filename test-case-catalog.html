Test Case Catalog

This document includes the test case description topics from the various test cases. It serves as a catalog of test cases.

If you create a new test case please add a reference to the test case description topic here.

The source for these test cases is available on GitHub as http://github.com/dita-community/dita-test-cases.

Note: This map also serves as a test of DITA 1.3 cross-deliverable links as it has cross-deliverable links to topics in the context of other root maps, either simply to relate the various test cases together or as a side effect of including test case description documents that themselves contain cross-deliverable links.
  • Key Space Tests
    Tests for various aspects of DITA key spaces and key references.
  • This-Topic Fragment ID Test 01
    Tests the resolution of this-topic fragment IDs within elements used by content reference.
  • Cross-Deliverable Link Test 01
    Tests the handling of cross-deliverable links using DITA 1.3's new meaning for scope="peer" and scoped key references. asdfasdj;lfm
  • Content Key Reference Tests
    Tests various configurations of referencing and referenced elements for content key reference.
